Flip!

A PWA created as an educational tool for users who are trying to learn using memorization. This app was created using the frameworks Vue.js and Node.js. It utilizes MongoDB and the ORM Mongoose.

Keepr

A social network that allows users to visually share, and discover new interests by posting (known as keeping) images to their own collections (vaults). It was created using Vue.js and .Net C#. It utilizies a MySQL DB and the Object Relational Mapper, Dapper.

Inspire Page

Built using Vanilla JavaScript, HTML and CSS, this is a page that utilizes Axios to import data from multiple web API's for weather and images. You can create and edit a To-Do list and also check out a Quote Of The Day.

Battle Cards Game

This a battle card game in which a player selects one of their cards and one of their opponents to fight. This game was created using the Vue framework to manage different states and pages. It also utlizies Axios to interact with the game's API.
